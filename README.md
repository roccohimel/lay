# Lay
prebult tmux session

How to install:

First, install htop and fastfetch
```
sudo apt install fastfetch htop # If on Debian
```
```
sudo dnf install fastfetch htop # If on RHEL
```
```
sudo pacman -Sy fastfetch htop # If on Arch
```
```
brew install fastfetch htop # If on MacOS
```
if this doesnt work, try bulding them from source.

Second, install Lay
```
bash install.sh
```
it should work.
